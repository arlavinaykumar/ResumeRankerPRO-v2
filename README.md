# ResumeRankerPRO v2

🎯 An AI-Powered Resume Ranking System using Python, SpaCy, Flask, and TF-IDF.

## 🚀 Features
- Upload multiple PDF resumes
- Compare against job description
- Rank resumes by match %
- Download HR-ready report

## 🛠 Tech Stack
- Python 3.13.2
- Flask
- SpaCy (`en_core_web_sm`)
- Scikit-learn
- PyMuPDF
